# pfSense Logs Integration - Syslog, Suricata and pfBlockerNG
---

## Configure syslog-ng pfSense package
In pfSense UI: 
- Go to **System > Package Manager > Available Packages**.
- Search for syslog-ng and install the package.
- Go to **Services > Syslog-ng > **General**.
- Set Interface Selection: loopback , Default Protocol: UDP, Default Port: 1514 (or choose any other port you want)

## Configure pfSense system logs forwarding
### System Logs (Built-in Syslog)
In pfSense UI:  
- Go to **Status > System Logs > Settings**.  
- Under **Remote Logging**, enable **Send log messages to remote syslog server**.  
- Add remote server: `127.0.0.1:1514` (UDP - The logs will be sent to syslog-ng).  
- Select categories to forward (e.g., Firewall, System, DHCP).  
- Save and Apply.  

## Configure syslog-ng Advanced objects for pfSense systems logs forwarding to vector:
In pfSense UI: 
- Go to **Services > Syslog-ng > Advanced**.  
- Add pfsense syslog destination object:
  1. Object Name: d_pfsense_syslog
  2. Object Type: Destination (the port assigned is the port we set that listening in vector.yaml for pfSense syslog)
  3. Object Parameters:
     ```
     { network("<YOUR-DEBIAN-IP>" transport(udp) port(5514)); };
     ```
- Add pfsense syslog log object:
  1. Object Name: log_pfsense_syslog
  2. Object Type: Log
  3. Object Parameters:
     ```
     { source(_DEFAULT); destination(d_pfsense_syslog);};
     ```
     
## Configure syslog-ng for Suricata:
**Assuming you have installed Suricata pfSense package and enabled EVE JSON Log in Suricata monitoring interface settings**
In pfSense UI: 
- Go to **Services > Syslog-ng > Advanced**.  
- Add suricata source object:
  1. Object Name: s_suricata
  2. Object Type: Source
  3. Objject Parameters:
     ```
     { wildcard-file( base-dir("/var/log/suricata") filename-pattern("*eve.json") recursive(yes) follow-freq(1) program-override("suricata") flags(no-parse) ); };
     ```
- Add suricata destination object:
  1. Object Name: d_suricata
  2. Object Type: Log
  3. Object Parameters:
     ```
     { network("YOUR-DEBIAN-IP" transport(udp) port(5140)); };
          
- Add suricata log object :
  1. Object Name: log_suricata
  2. Object Type: Log
  3. Object Parameters:
     ```
     { source(s_suricata); destination(d_suricata); };
     ```
         
- Restart syslog-ng service in pfSense
```
service syslog-ng restart

# If error
killall syslog-ng
service syslog-ng start
```

## Configure pfSense cron job for pfBlockerNG unified.log:
**Assuming you have installed pfBlockerNG pfSense package and enabled feeds.**

### Create /var/log/pfblockerng/ directory in Debian
```
sudo mkdir -p /var/log/pfblockerng/
```

### Set up SSH key auth from pfSense to Debian (no password prompt)
In pfSense UI: 
- Go to **Diagnostics > Command Prompt
Execute
```
ssh-keygen -t ed25519 -N "" -f /root/.ssh/pfblocker_sync
```

Copy public key to Debian authorized keys.

Skip prompt
```
ssh-keyscan <YOUR-DEBIAN-IP> >> /root/.ssh/known_hosts 2>/dev/null
```

Test connection
```
ssh -i /root/.ssh/pfblocker_sync debian-userx@<YOUR-DEBIAN-IP> "echo ok"
```

Copy unified.log file to /var/log/pfblockerng
```
scp -i /root/.ssh/pfblocker_sync /var/log/pfblockerng/unified.log debia-user@<YOUR-DEBIAN-IP>:/var/log/pfblockerng/unified.log
```

## Set Vector permissions for /var/log/pfblockerng
```
sudo chown debian-user:vector /var/log/pfblockerng
sudo chmod 750 /var/log/pfblockerng
sudo chmod 640 /var/log/pfblockerng/unified.log
sudo chgrp vector /var/log/pfblockerng/unified.log
```

### Create pfsene cron job to sync every 60 seconds and keep the group readable in future scp transfers
In pfSense UI: 
- Go to Services > Cron > Add
Schedule: * * * * *
User: root
Command:
```
/usr/bin/scp -i /root/.ssh/pfblocker_sync -o StrictHostKeyChecking=no /var/log/pfblockerng/unified.log debian-user@<YOUR-DEBIAN-IP>:/var/log/pfblockerng/unified.log && /usr/bin/ssh -i /root/.ssh/pfblocker_sync debian-user@<YOUR-DEBIAN-IP> "chmod 640 /var/log/pfblockerng/unified.log && chgrp vector /var/log/pfblockerng/unified.log"
```

## Configure Vector for pfSense Logs
Vector will receive: standard pfSense system logs on 5514 (UDP) and Suricata EVE JSON on 5140 (UDP).

## Create Vector Pipeline
Update the file in `/etc/vector/vector.yaml` . Replace the existing file content with below:  
```
data_dir: /var/lib/vector
sources:
  suricata_pfsense:
    type: syslog
    mode: udp
    address: "0.0.0.0:5140"

  pfsense_syslog:
    type: syslog
    mode: udp
    address: "0.0.0.0:5514"
    max_message_size: 102400

  pfblocker_file:
    type: file
    include:
      - /var/log/pfblockerng/unified.log
    read_from: end
    ignore_checkpoints: false

  wazuh_alerts:
    type: file
    include:
      - /var/ossec/logs/alerts/alerts.json
    read_from: end
    ignore_checkpoints: false

transforms:
  suricata_parse_json:
    type: remap
    inputs: ["suricata_pfsense"]
    source: |
      structured, err = parse_json(string!(.message))
      if err != null {
        abort
      }
      . = structured
      .suricata.eve = .
      del(.suricata.eve.suricata)

  suricata_main:
    type: remap
    inputs: ["suricata_parse_json"]
    source: |
      .event.ingested = now()

      if exists(.suricata.eve.ether.dest_mac) {
        .destination.mac = del(.suricata.eve.ether.dest_mac)
      }
      if exists(.suricata.eve.ether.src_mac) {
        .source.mac = del(.suricata.eve.ether.src_mac)
      }

      if exists(.suricata.eve.event_type) {
        .suricata.eve.event_type = downcase(string!(.suricata.eve.event_type))
      }

      .event.kind = "event"
      .event.category = ["network"]

      event_type = to_string(.suricata.eve.event_type) ?? ""

      if event_type == "dns" {
        .event.type = ["protocol"]
        .network.protocol = "dns"
      } else if event_type == "http" || event_type == "http2" {
        .event.category = ["network", "web"]
        .event.type = ["access", "protocol"]
        .network.protocol = "http"
      } else if event_type == "flow" {
        .event.type = ["connection"]
      } else if event_type == "ftp_data" {
        .event.type = ["protocol"]
        .network.protocol = "ftp"
      } else if event_type == "stats" {
        .event.kind = "metric"
      } else if event_type == "rdp" || event_type == "rfb" {
        .event.type = ["protocol"]
        .network.protocol = "rdp"
      } else if event_type == "tftp" {
        .event.type = ["protocol"]
        .network.protocol = "tftp"
      } else if event_type == "mqtt" {
        .event.type = ["protocol"]
        .network.protocol = "mqtt"
      } else if event_type == "smb" {
        .event.type = ["protocol"]
        .network.protocol = "smb"
      } else if event_type == "ssh" {
        .event.type = ["protocol"]
        .network.protocol = "ssh"
      } else if event_type == "alert" {
        .event.kind = "alert"
        .event.category = ["network", "intrusion_detection"]
      } else if event_type == "tls" {
        .event.type = ["protocol"]
        .network.protocol = "tls"
      } else if event_type == "ftp" {
        .event.type = ["protocol"]
        .network.protocol = "ftp"
      }

      if exists(.suricata.eve.app_proto) {
        .suricata.eve.app_proto = downcase(string!(.suricata.eve.app_proto))
        app_proto = string(.suricata.eve.app_proto)
        if app_proto == "ftp-data" {
          .network.protocol = "ftp"
        } else if app_proto != "failed" && app_proto != "template" && app_proto != "template-rust" && app_proto != "" {
          .network.protocol = app_proto
        }
      }

      if event_type == "http" {
        status = to_int(.suricata.eve.http.status) ?? -1
        if status >= 0 {
          if status < 400 {
            .event.outcome = "success"
          } else {
            .event.outcome = "failure"
          }
        }
      }

      flow_state = to_string(.suricata.eve.flow.state) ?? ""
      if !exists(.event.type) || !is_array(.event.type) { .event.type = [] }
      if flow_state == "new" {
        .event.type = push!(.event.type, "start")
      } else if flow_state == "closed" {
        .event.type = push!(.event.type, "end")
      }

      if exists(.suricata.eve.fileinfo.filename) {
        .file.path = del(.suricata.eve.fileinfo.filename)
      }
      if exists(.suricata.eve.fileinfo.size) {
        .file.size = del(.suricata.eve.fileinfo.size)
      }

      if exists(.network.transport) {
        .network.transport = downcase(string!(.network.transport))
      }

      if exists(.suricata.eve.alert.category) {
        .message = to_string!(.suricata.eve.alert.category)
        .rule.category = to_string!(.suricata.eve.alert.category)
      }
      if exists(.suricata.eve.alert.signature_id) {
        .rule.id = to_string!(.suricata.eve.alert.signature_id)
      }
      if exists(.suricata.eve.alert.signature) {
        .rule.name = to_string!(.suricata.eve.alert.signature)
      }

      if exists(.suricata.eve.alert.action) {
        action = to_string!(.suricata.eve.alert.action)
        if action == "blocked" {
          action = "denied"
          .suricata.eve.alert.action = "denied"
        }
        if !exists(.event.type) || !is_array(.event.type) { .event.type = [] }
        .event.type = push!(.event.type, action)
        del(.suricata.eve.alert.action)
      }

      if exists(.suricata.eve.alert.severity) {
        .event.severity = del(.suricata.eve.alert.severity)
      }

      if exists(.suricata.eve.alert.metadata.protocols) {
        .suricata.eve.alert.protocols = del(.suricata.eve.alert.metadata.protocols)
      }
      if exists(.suricata.eve.alert.metadata.attack_target) {
        .suricata.eve.alert.attack_target = del(.suricata.eve.alert.metadata.attack_target)
      }
      if exists(.suricata.eve.alert.metadata.capec_id) {
        .suricata.eve.alert.capec_id = del(.suricata.eve.alert.metadata.capec_id)
      }
      if exists(.suricata.eve.alert.metadata.cwe_id) {
        .suricata.eve.alert.cwe_id = del(.suricata.eve.alert.metadata.cwe_id)
      }
      if exists(.suricata.eve.alert.metadata.malware) {
        .suricata.eve.alert.malware = del(.suricata.eve.alert.metadata.malware)
      }
      if exists(.suricata.eve.alert.metadata.cve) {
        .suricata.eve.alert.cve = del(.suricata.eve.alert.metadata.cve)
      }
      if exists(.suricata.eve.alert.metadata.cvss_v2_base) {
        .suricata.eve.alert.cvss_v2_base = del(.suricata.eve.alert.metadata.cvss_v2_base)
      }
      if exists(.suricata.eve.alert.metadata.cvss_v2_temporal) {
        .suricata.eve.alert.cvss_v2_temporal = del(.suricata.eve.alert.metadata.cvss_v2_temporal)
      }
      if exists(.suricata.eve.alert.metadata.cvss_v3_base) {
        .suricata.eve.alert.cvss_v3_base = del(.suricata.eve.alert.metadata.cvss_v3_base)
      }
      if exists(.suricata.eve.alert.metadata.cvss_v3_temporal) {
        .suricata.eve.alert.cvss_v3_temporal = del(.suricata.eve.alert.metadata.cvss_v3_temporal)
      }
      if exists(.suricata.eve.alert.metadata.priority) {
        .suricata.eve.alert.priority = del(.suricata.eve.alert.metadata.priority)
      }
      if exists(.suricata.eve.alert.metadata.hostile) {
        .suricata.eve.alert.hostile = del(.suricata.eve.alert.metadata.hostile)
      }
      if exists(.suricata.eve.alert.metadata.infected) {
        .suricata.eve.alert.infected = del(.suricata.eve.alert.metadata.infected)
      }
      if exists(.suricata.eve.alert.metadata.filename) {
        .file.name = del(.suricata.eve.alert.metadata.filename)
      }
      if exists(.suricata.eve.alert.metadata.classtype) {
        .suricata.eve.alert.classtype = del(.suricata.eve.alert.metadata.classtype)
      }
      if exists(.suricata.eve.alert.metadata.rule_source) {
        .suricata.eve.alert.rule_source = del(.suricata.eve.alert.metadata.rule_source)
      }
      if exists(.suricata.eve.alert.metadata.sid) {
        .suricata.eve.alert.sid = del(.suricata.eve.alert.metadata.sid)
      }

      if exists(.suricata.eve.alert.metadata.created_at) {
        created_at_raw = del(.suricata.eve.alert.metadata.created_at)
        created_at_str = if is_array(created_at_raw) {
          join!(array!(created_at_raw), ",")
        } else {
          to_string!(created_at_raw)
        }
        parsed_ca, err = parse_timestamp(created_at_str, "%Y-%m-%d")
        if err != null {
          parsed_ca, err = parse_timestamp(created_at_str, "%Y_%m_%d")
        }
        if err == null {
          .suricata.eve.alert.created_at = parsed_ca
        }
      }

      if exists(.suricata.eve.alert.metadata.updated_at) {
        updated_at_raw = del(.suricata.eve.alert.metadata.updated_at)
        updated_at_str = if is_array(updated_at_raw) {
          join!(array!(updated_at_raw), ",")
        } else {
          to_string!(updated_at_raw)
        }
        parsed_ua, err = parse_timestamp(updated_at_str, "%Y-%m-%d")
        if err != null {
          parsed_ua, err = parse_timestamp(updated_at_str, "%Y_%m_%d")
        }
        if err == null {
          .suricata.eve.alert.updated_at = parsed_ua
        }
      }

      if exists(.suricata.eve.alert.metadata.mitre_attack) {
        .threat.tactic.id = del(.suricata.eve.alert.metadata.mitre_attack)
      }
      if exists(.suricata.eve.alert.metadata.mitre_tactic_id) && !exists(.threat.tactic.id) {
        .threat.tactic.id = del(.suricata.eve.alert.metadata.mitre_tactic_id)
      }
      if exists(.suricata.eve.alert.metadata.mitre_tactic_name) {
        .threat.tactic.name = del(.suricata.eve.alert.metadata.mitre_tactic_name)
      }
      if exists(.suricata.eve.alert.metadata.mitre_technique_id) {
        .threat.technique.id = del(.suricata.eve.alert.metadata.mitre_technique_id)
        tech_id = .threat.technique.id
        if is_array(tech_id) {
          .threat.technique.id = get!(tech_id, [0])
        }
      .threat.technique.name = to_string!(.threat.technique.id)
      }

      if exists(.suricata.eve.flow.pkts_toclient) {
        .destination.packets = del(.suricata.eve.flow.pkts_toclient)
      }
      if exists(.suricata.eve.flow.pkts_toserver) {
        .source.packets = del(.suricata.eve.flow.pkts_toserver)
      }
      if exists(.suricata.eve.flow.bytes_toclient) {
        .destination.bytes = del(.suricata.eve.flow.bytes_toclient)
      }
      if exists(.suricata.eve.flow.bytes_toserver) {
        .source.bytes = del(.suricata.eve.flow.bytes_toserver)
      }

      sb = to_int(.source.bytes) ?? 0
      db = to_int(.destination.bytes) ?? 0
      sp = to_int(.source.packets) ?? 0
      dp = to_int(.destination.packets) ?? 0
      if (sb + db) > 0 {
        .network.bytes = sb + db
      }
      if (sp + dp) > 0 {
        .network.packets = sp + dp
      }

      if exists(.suricata.eve.flow.start) {
        parsed_start, err = parse_timestamp(string!(.suricata.eve.flow.start), "%+")
        if err == null {
          .event.start = parsed_start
        }
      }
      if exists(.suricata.eve.flow.end) {
        parsed_end, err = parse_timestamp(string!(.suricata.eve.flow.end), "%+")
        if err == null {
          .event.end = parsed_end
        }
      }

      if exists(.event.start) && exists(.event.end) {
        start_ns = to_unix_timestamp!(.event.start, unit: "nanoseconds")
        end_ns = to_unix_timestamp!(.event.end, unit: "nanoseconds")
        if end_ns >= start_ns {
          .event.duration = end_ns - start_ns
        }
      }

      if exists(.suricata.eve.proto) {
        .network.transport = downcase(string!(.suricata.eve.proto))
      }

      if !exists(.related.ip) || !is_array(.related.ip) { .related.ip = [] }
      if exists(.source.ip) {
        .related.ip = push!(.related.ip, .source.ip)
      }
      if exists(.destination.ip) {
        .related.ip = push!(.related.ip, .destination.ip)
      }

      if exists(.url.domain) {
        url_domain_str = to_string(.url.domain) ?? ""
        if url_domain_str != "" {
          if !exists(.related.hosts) || !is_array(.related.hosts) { .related.hosts = [] }
          .related.hosts = push!(.related.hosts, .url.domain)
        }
      }

      del(.suricata.eve.app_proto)
      del(.suricata.eve.flow.end)
      del(.suricata.eve.flow.start)
      del(.suricata.eve.http.http_method)
      del(.suricata.eve.http.http_user_agent)

  suricata_dns:
    type: remap
    inputs: ["suricata_main"]
    source: |
      proto = to_string(.network.protocol) ?? ""
      if proto == "dns" {
        if exists(.suricata.eve.dns.id) {
          .dns.id = to_string!(.suricata.eve.dns.id)
        }
        if exists(.suricata.eve.dns.rcode) {
          .dns.response_code = to_string!(.suricata.eve.dns.rcode)
        }
        if exists(.suricata.eve.dns.type) {
          .dns.type = to_string!(.suricata.eve.dns.type)
        }

        dns_type = to_string(.suricata.eve.dns.type) ?? ""
        dns_version = to_int(.suricata.eve.dns.version) ?? 0

        if dns_type == "query" || dns_version == 2 {
          if exists(.suricata.eve.dns.rrname) {
            .dns.question.name = to_string!(.suricata.eve.dns.rrname)
          }
          if exists(.suricata.eve.dns.rrtype) {
            .dns.question.type = to_string!(.suricata.eve.dns.rrtype)
          }
        }

        if dns_type == "answer" {
          if dns_version == 0 {
            answer = {}
            if exists(.suricata.eve.dns.rrname) {
              answer.name = to_string!(.suricata.eve.dns.rrname)
            }
            if exists(.suricata.eve.dns.rdata) {
              answer.data = to_string!(.suricata.eve.dns.rdata)
            }
            if exists(.suricata.eve.dns.rrtype) {
              answer.type = to_string!(.suricata.eve.dns.rrtype)
            }
            if exists(.suricata.eve.dns.ttl) {
              answer.ttl = .suricata.eve.dns.ttl
            }
            .dns.answers = [answer]

            rtype_v1 = to_string(.suricata.eve.dns.rrtype) ?? ""
            if rtype_v1 == "A" || rtype_v1 == "AAAA" {
              rdata = to_string(.suricata.eve.dns.rdata) ?? null
              if rdata != null {
                .dns.resolved_ip = [rdata]
              }
            }
          } else if dns_version == 2 {
            if exists(.suricata.eve.dns.answers) {
              raw_answers = array!(.suricata.eve.dns.answers)
              resolved_ips = []
              normalized = []
              for_each(raw_answers) -> |_index, ans| {
                entry = {}
                ans_obj = object!(ans)
                n = get(ans_obj, ["rrname"]) ?? null
                t = get(ans_obj, ["rrtype"]) ?? null
                d = get(ans_obj, ["rdata"]) ?? null
                ttl = get(ans_obj, ["ttl"]) ?? null
                if n != null { entry.name = to_string!(n) }
                if t != null { entry.type = to_string!(t) }
                if d != null { entry.data = to_string!(d) }
                if ttl != null { entry.ttl = ttl }
                if t != null && d != null {
                  rtype_v2 = to_string!(t)
                  if rtype_v2 == "A" || rtype_v2 == "AAAA" {
                    resolved_ips = push(resolved_ips, to_string!(d))
                  }
                }
                normalized = push(normalized, entry)
              }
              .dns.answers = normalized
              del(.suricata.eve.dns.answers)
              if length(resolved_ips) > 0 {
                .dns.resolved_ip = resolved_ips
              }
            }
          }
        }

        if exists(.dns.resolved_ip) {
          if !exists(.related.ip) || !is_array(.related.ip) { .related.ip = [] }
          for_each(array!(.dns.resolved_ip)) -> |_index, ip| {
            .related.ip = push!(.related.ip, ip)
          }
        }

        if exists(.dns.question.registered_domain) {
          rd = string!(.dns.question.registered_domain)
          rd_parts = split(rd, ".")
          if length(rd_parts) > 1 {
            tld_parts = slice!(rd_parts, 1)
            .dns.question.top_level_domain = join!(tld_parts, ".")
          }
        }

        if !exists(.dns.header_flags) || !is_array(.dns.header_flags) { .dns.header_flags = [] }
        if to_bool(.suricata.eve.dns.aa) ?? false {
          .dns.header_flags = push!(.dns.header_flags, "AA")
        }
        if to_bool(.suricata.eve.dns.tc) ?? false {
          .dns.header_flags = push!(.dns.header_flags, "TC")
        }
        if to_bool(.suricata.eve.dns.rd) ?? false {
          .dns.header_flags = push!(.dns.header_flags, "RD")
        }
        if to_bool(.suricata.eve.dns.ra) ?? false {
          .dns.header_flags = push!(.dns.header_flags, "RA")
        }

        del(.suricata.eve.dns.aa)
        del(.suricata.eve.dns.tc)
        del(.suricata.eve.dns.rd)
        del(.suricata.eve.dns.ra)
        del(.suricata.eve.dns.qr)
        del(.suricata.eve.dns.version)
        del(.suricata.eve.dns.flags)
        del(.suricata.eve.dns.grouped)
      }

  suricata_tls:
    type: remap
    inputs: ["suricata_dns"]
    source: |
      proto = to_string(.network.protocol) ?? ""
      if proto == "tls" {
        tls_ver_raw = to_string(.suricata.eve.tls.version) ?? ""
        if tls_ver_raw != "UNDETERMINED" && tls_ver_raw != "" {
          ver_parts = split(tls_ver_raw, " ")
          if length(ver_parts) == 2 {
            .tls.version_protocol = downcase(to_string(ver_parts[0]))
            .tls.version = to_string(ver_parts[1])
          }
        }

        if exists(.suricata.eve.tls.sni) {
          sni = string!(.suricata.eve.tls.sni)
          if ends_with(sni, ".") {
            .suricata.eve.tls.sni = slice!(sni, 0, length(sni) - 1)
          }
        }

        if exists(.suricata.eve.tls.subject) {
          subj_str = string!(.suricata.eve.tls.subject)
          .tls.server.subject = subj_str
          subj_clean = replace(subj_str, "\\,", "")
          subj_parts = split(subj_clean, ", ")
          for_each(subj_parts) -> |_index, part| {
            kv = split(to_string(part), "=")
            if length(kv) >= 2 {
              k = strip_whitespace(to_string(kv[0]))
              v = strip_whitespace(join!(slice!(kv, 1), "="))
              if k == "C"  { .tls.server.x509.subject.country             = v }
              if k == "CN" { .tls.server.x509.subject.common_name         = v }
              if k == "L"  { .tls.server.x509.subject.locality            = v }
              if k == "O"  { .tls.server.x509.subject.organization        = v }
              if k == "OU" { .tls.server.x509.subject.organizational_unit = v }
              if k == "ST" { .tls.server.x509.subject.state_or_province   = v }
            }
          }
        }

        if exists(.suricata.eve.tls.issuerdn) {
          issuer_str = string!(.suricata.eve.tls.issuerdn)
          .tls.server.issuer = issuer_str
          issuer_clean = replace(issuer_str, "\\,", "")
          issuer_parts = split(issuer_clean, ", ")
          for_each(issuer_parts) -> |_index, part| {
            kv = split(to_string(part), "=")
            if length(kv) >= 2 {
              k = strip_whitespace(to_string(kv[0]))
              v = strip_whitespace(join!(slice!(kv, 1), "="))
              if k == "C"  { .tls.server.x509.issuer.country             = v }
              if k == "CN" { .tls.server.x509.issuer.common_name         = v }
              if k == "L"  { .tls.server.x509.issuer.locality            = v }
              if k == "O"  { .tls.server.x509.issuer.organization        = v }
              if k == "OU" { .tls.server.x509.issuer.organizational_unit = v }
              if k == "ST" { .tls.server.x509.issuer.state_or_province   = v }
            }
          }
        }

        if exists(.suricata.eve.tls.session_resumed) {
          .tls.resumed = to_bool!(.suricata.eve.tls.session_resumed)
        }

        if exists(.suricata.eve.tls.fingerprint) {
          sha1 = upcase(string!(.suricata.eve.tls.fingerprint))
          sha1 = replace(sha1, ":", "")
          .tls.server.hash.sha1 = sha1
          if !exists(.related.hash) || !is_array(.related.hash) { .related.hash = [] }
          .related.hash = push!(.related.hash, sha1)
        }

        if exists(.suricata.eve.tls.sni) {
          sni_val = string!(.suricata.eve.tls.sni)
          if sni_val != "" {
            .tls.client.server_name = sni_val
            .destination.domain = sni_val
          }
        }

        if exists(.suricata.eve.tls.ja3s.hash) {
          .tls.server.ja3s = to_string!(.suricata.eve.tls.ja3s.hash)
        }
        if exists(.suricata.eve.tls.ja3.hash) {
          .tls.client.ja3 = to_string!(.suricata.eve.tls.ja3.hash)
        }

        if exists(.suricata.eve.tls.certificate) {
          .tls.server.certificate = string!(.suricata.eve.tls.certificate)
        }
        if exists(.suricata.eve.tls.chain) {
          .tls.server.certificate_chain = .suricata.eve.tls.chain
        }

        if exists(.suricata.eve.tls.serial) {
          .tls.server.x509.serial_number = replace(string!(.suricata.eve.tls.serial), ":", "")
        }

        if exists(.suricata.eve.tls.notafter) {
          parsed_na, err = parse_timestamp(string!(.suricata.eve.tls.notafter), "%+")
          if err == null {
            .tls.server.not_after = parsed_na
            .tls.server.x509.not_after = parsed_na
          }
        }
        if exists(.suricata.eve.tls.notbefore) {
          parsed_nb, err = parse_timestamp(string!(.suricata.eve.tls.notbefore), "%+")
          if err == null {
            .tls.server.not_before = parsed_nb
            .tls.server.x509.not_before = parsed_nb
          }
        }
      }

            .related.hosts = push!(.related.hosts, .url.domain)
            .related.hosts = unique(array(.related.hosts))

  pfsense_filterlog:
    type: remap
    inputs: ["pfsense_syslog"]
    source: |
      if !exists(.appname) || downcase(string!(.appname)) != "filterlog" { abort }
      msg = string!(.message)
      if !match(msg, r'^\d+,') { abort }
      cols = parse_csv!(msg)
      if length(cols) < 22 { abort }
      rule_s     = string!(cols[0])
      subrule_s  = string!(cols[1])
      iface      = string!(cols[4])
      reason     = string!(cols[5])
      action     = string!(cols[6])
      direction  = string!(cols[7])

      length_s   = string!(cols[17])
      src_ip     = string!(cols[18])
      dst_ip     = string!(cols[19])
      src_port_s = string!(cols[20])
      dst_port_s = string!(cols[21])
      .pfsense.iface     = iface
      .pfsense.reason    = reason
      .pfsense.action    = action
      .pfsense.direction = direction
      .event.action      = downcase(action)
      .network.direction = downcase(direction)
      if match(rule_s, r'^\d+$')     { .pfsense.rule    = to_int!(rule_s) }
      if match(subrule_s, r'^\d+$')  { .pfsense.subrule = to_int!(subrule_s) }
      if match(ipver_s, r'^\d+$')    { .network.iana_number = to_int!(ipver_s) }
      if match(ttl_s, r'^\d+$')      { .pfsense.ttl     = to_int!(ttl_s) }
      if match(length_s, r'^\d+$')   { .network.bytes   = to_int!(length_s) }
      if match(proto_id_s, r'^\d+$') {
        .pfsense.proto_id = to_int!(proto_id_s)
      } else if proto_id_s != "" {
        .pfsense.proto_id_raw = proto_id_s
      }
      if match(src_port_s, r'^\d+$') { .source.port      = to_int!(src_port_s) }
      if match(dst_port_s, r'^\d+$') { .destination.port = to_int!(dst_port_s) }
      .source.ip      = src_ip
      .destination.ip = dst_ip
      if proto_s != "" { .network.transport = downcase(proto_s) }
      if match(ipver_s, r'^[46]$') { .network.type = if ipver_s == "4" { "ipv4" } else { "ipv6" } }
      .event.category   = ["network"]
      .event.type       = ["connection"]
      .event.kind       = "event"
      .event.dataset    = "pfsense.filterlog"
      .event.module     = "pfsense"
      .observer.product = "pfSense"
      .observer.vendor  = "Netgate"
      .observer.type    = "firewall"
      if exists(.host) { .observer.hostname = del(.host) }
      del(.message)
      del(.appname)
      del(.facility)
      del(.severity)
      del(.procid)
      del(.msgid)
      del(.version)
      if !exists(.tags) || !is_array(.tags) { .tags = [] }
      .tags = push!(.tags, "source_pfsense")
